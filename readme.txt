程序概述：

1.本程序的开发平台是Visual Studio 2017

2.程序自带Log4net组件，并支持Log日志书写功能

3.程序自带所需dll文件

4.暂用Data中的Userinfo.xml作为数据库，自动配置，无需额外操作


功能概述：

1.本程序用xml文件模拟数据库，可以进行增加、删除、查找、保存操作，点击相应的按钮并按照程序提示即可

2.生成日志的目录是工程路径下bin\Debug，在log.txt中记录了操作内容
